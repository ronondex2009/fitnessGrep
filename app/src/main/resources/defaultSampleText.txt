This file contains default sample text that WordFitness.java will use when calling FromSample() with no arguments.
This file orginially contained temporary sample text which has been purged since it was not good sample text.
Also, this text file is deprecated. FromSample() with no arguments is also deprecated.

Instead, users should use FromSample(Scanner sc) on their own sample file. If using defaults, users should then use
FromFrequencies() with no arguments instead. This is the most preferred way to build the WordFitness class.

This file and the FromSample() without arguments builder may be removed in the future.
